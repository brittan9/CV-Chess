
Basketball-Hoop-Detection - v3 VancouverHoops1
==============================

This dataset was exported via roboflow.ai on May 24, 2021 at 3:02 AM GMT

It includes 813 images.
Hoops are annotated in YOLO v5 PyTorch format.

The following pre-processing was applied to each image:
* Resize to 544x544 (Stretch)

The following augmentation was applied to create 3 versions of each source image:
* Random rotation of between -2 and +2 degrees
* Random exposure adjustment of between -15 and +15 percent
